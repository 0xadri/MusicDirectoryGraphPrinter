

Manual:

	1. create the .exe file (mvn package from the app's root dir)
	1. drop the .exe file it in the directory you want to analyse
	2. execute the file
	3. it will create the file "directorygraph-yyyy-mm-dd-HH-SS.txt" in the same directory. Check the content.
